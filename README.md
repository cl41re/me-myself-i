# danialazn.dev

Personal portfolio built with Astro, React, and Tailwind CSS.

Live: [danialazn.dev](https://danialazn.dev)

## About

This project is my personal website. It is designed to feel minimal, atmospheric, and motion-led while still staying technically clean under the hood.

The current build focuses on a cinematic hero section, lightweight copy, and interactive visual texture through custom components like `LaserFlow`, `ScrambledText`, and `DecryptedText`.

## Stack

- Astro
- React
- Tailwind CSS
- Vercel
- Cloudflare DNS

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Notes

- Custom domain configured through Cloudflare DNS
- Deployed on Vercel
- Built as a personal space for frontend work, interaction, and visual direction
